Errata

There have been relatively few reports of errors in the book. Thanks to Dmitry Moiseev for spotting these.

In some editions of the book (I presume the Global edition) there is an error on the very last line of section 4.5.3 where ‘Alternatively’ has been misspelled. This is not the case for the 2020 edition, printed in the US.

On page 236 of the 2020 edition, Table 8.2 has a duplicate entry for the Facade pattern. Essentially, these same the same thing. Obviously, I rewrote it but forgot to delete the original entry.
